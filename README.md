# 🚀 Projeto AWS - Redução de Custos em Ambiente Farmacêutico

## 👨‍💻 Autor

**Higor Padilha**



## 📌 Sobre o Projeto

Este projeto apresenta a implementação estratégica de serviços da AWS em uma empresa farmacêutica de pequeno porte (**Abstergo Industries**), com foco em **redução imediata de custos**, **escalabilidade** e **automação de processos**.

A solução foi pensada para substituir infraestrutura local cara por serviços em nuvem altamente eficientes.



## 🎯 Objetivo

* Reduzir custos operacionais com infraestrutura
* Melhorar desempenho dos sistemas internos
* Garantir segurança e disponibilidade dos dados
* Automatizar processos repetitivos





## ☁️ Serviços AWS Utilizados

### 📦 Amazon S3

* Armazenamento de documentos laboratoriais e regulatórios
* Alta durabilidade e baixo custo
* Eliminação de servidores físicos

### 💻 Amazon EC2

* Execução de sistemas internos (estoque, produção)
* Escalabilidade sob demanda
* Redução de custos com hardware

### ⚡ AWS Lambda

* Automação de tarefas (relatórios, atualizações)
* Execução sob demanda (pay-per-use)
* Redução de processos manuais




## 🏗️ Arquitetura da Solução

* Dados armazenados no S3
* Aplicações rodando em EC2
* Processos automatizados com Lambda

*(Arquitetura simples, escalável e de baixo custo)*




## 💰 Resultados Obtidos

* ✅ Redução de custos com infraestrutura física
* ✅ Maior segurança dos dados
* ✅ Processos automatizados
* ✅ Escalabilidade para crescimento futuro




## 📊 Diferenciais do Projeto

* Aplicação prática de cloud computing
* Foco em **problema real de negócio (custo)**
* Arquitetura simples e eficiente
* Pensado para pequenas empresas
